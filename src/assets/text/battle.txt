Available Actions:

~ this.attack(target); ~
perform a standard attack on the specified target (scroll to view target list).

~ this.cast(spell, target); ~
cast the specified spell on the specified target (scroll to view spell and target lists).

Available Variables:

~ this.warriorTurn ~
returns true or false if it is your warrior's turn

~ this.mageTurn ~
returns true or false if it is your mage's turn

~ this.rangerTurn ~
returns true or false if it is your ranger's turn

~ this.turn ~
returns an integer representing the current turn number

Available Spells:

<<< Black Magic >>>
"earth", "dark", "fire",  "lighting", "water", "wind"

<<< White Magic >>>
"cure", "protect", "remedy", "raise"

Available Targets:

"warrior name"
"mage name"
"ranger name"
"Enemy1"
"Enemy2"
"Enemy3"
"random" - chooses a random enemy
